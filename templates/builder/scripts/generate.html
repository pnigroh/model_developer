CSV Format

Semicolon-separated, first row is the header:

Class;Field;DataType;Summary
Property;Title;Text;
Property;Price;Money;
Property;Status;Text;
PropertyImage;property;Property;FK to Property

The script creates:
  • __init__.py
  • models.py — all model classes
  • admin.py — @register stubs
  • apps.py — AppConfig
